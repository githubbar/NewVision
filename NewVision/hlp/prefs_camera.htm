Camera Model

This dialog controls camera calibration for your tracking project.

1. Main Screen

This screen allows to calibrate the camera "Monoscopic" mode
The calibration is done in two steps:
  1. Setting the vanishing points:
    1. Find at least two parallel lines in the scence (buildings, isles, boxes, etc) and with the left mouse button drag-and-draw RED parallel lines on top. These line will represent parallel lines in the X-axis direction. You can drag the endpoints of the line to adjust its position. While dragging, pressing DELETE button will remove the line.
    2. Now do the same for Y-axis direction this time using the middle mouse button (or pressing on the mouse wheel) and GREEN lines.
    3. Find parallel lines in the scence going vertically up from the ground. Now do the same for Z-axis direction this time using the right mouse button and BLUE lines. This step is essential to determining the correct height of the objects.

  2. Setting the scale:
    1. This time hold the CTRL key and with any mouse button, mark at least two vertical lines of know height. These lines will appear as LIGHT BLUE. Once done enter the height of each line (in cm) in "Reference heights" box on the right.
Default setting:

2. Camera Type

Currently two camera models are supported: panoramic "unprojected map" model and "perspective projection" (regular monocular camera) model. If camera type is set to "panoramic" the dialog will change prompting you to enter only the elevation of the camera from the ground (cm).


Default setting: Monoscopic

3. Reference heights

Specify the height of each reference LIGHT BLUE line (in cm). Click "Add" to add values and "Delete" to clear all values.
Default setting: None

4. Reset

Clears all the settings, including the main screen.
Default setting: None
