DoorModel

This dialog controls how the doors are represented in your tracking project.

1. Main screen

Use left mouse button to place individual door points or draw an "interest region" polygon. You can drag the door to adjust their position. While dragging, pressing DELETE button will remove the door.

2. Remove all

Removes all doors
Default setting: None

3. Door model type

Select how the doors are represented in the system. The doors are indicating where pedestrians are entering/leaving the scene.
  • Uniform: each point in space is equally likely to be a door
  • Points: doors ar X, Y points on the floor
  • Interest region: door are located along the perimeter of a polygonal interest region

Default setting: Uniform
