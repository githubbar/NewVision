File menu commands

The File menu offers the following commands:

New Project Creates a new project.
Open Project Opens an existing document.
Save Project Saves an opened project using the same file name.
Save Project As Saves an opened project to a specified file name.
Print Prints current project settings.
Print Preview Displays the current project settings on the screen as they would appear printed.
Print Setup Selects a printer and printer connection.
Exit Exits New Vision™.
