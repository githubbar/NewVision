FloorModel

This dialog controls the way people paths are displayed in the floor view.

1. Main screen

Use left mouse button to designate the displayable floor area by specifying extreme floor locations. You can specify as many points as you want. You can drag the points to adjust their position. While dragging, pressing DELETE button will remove the point.

2. Remove all

Removes all extreme points on the floor
Default setting: None

3. Select floor model

  • Image: superimpose people tracks on to any BMP image (click "Browse" to select one)
  • CAD model: use computer generated floor model

Default setting: Image

2. Floor size

Specify the size of the floor image in pixels (must be the same size as the BMP you loaded)
Default setting: 800x600
