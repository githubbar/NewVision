Input Model

This dialog controls the input source for your tracking project.

1. Select input source

The tracker can take input from either a color video file, from a camera connected to your computer or from thermal and color camera inputs combined in a single video file (stitched horizontally)
Default setting: File

2. Offset (frames)

If you are using a video file as your input, you can specify to start tracking from any frame in the file. Typical videos have a framerate of 30 FPS so, for example, to start tracking 30 seconds after the sequence begins type in 900 (30*30=900).
Default setting: 0

3. Start date and time

You can specify the exact date and time when the recording took place. This can be later used to display correct date and time on top of the video.
Default setting: Current date and time
