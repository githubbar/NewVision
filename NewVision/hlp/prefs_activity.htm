Activity Model

This dialog controls people activity recognition for your tracking project.

1. Group is closer than

Experimental. Used to determine groups of people
Default setting: 5000

2. Sales rep. histogram correlation

Experimental
Default setting: 0.9

3. Smooth paths by

Average the coordinates from the previous locations to smooth people trajectories.
Default setting: 0.3

4. Dwell time

Specify for how long a person has to be within the dwell area to be marked as dwelling
Default setting: 4

5. Dwell area

The span (+/-) of the dwell area. Example: If this is set to 50 the area becomes a circle of radius 50 cm
Default setting: 100
