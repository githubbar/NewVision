Data menu commands

The Data menu offers the following commands:

Save Heat Map

Save heat map data in XML format. (easily convertible to CSV)

Save Floor View

Save Floor View as a BMP image

Save Left View

Save Left View as a BMP image

Save Right View

Save Right View as a BMP image

Load Body Paths

Load tracked body information from a DAT file stored in New Vision™ internal format.

Save Body Paths

Saved tracked body information to a DAT file in New Vision™ internal format.
